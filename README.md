# Forge Auto Works

A modern, fully responsive static website for a fictional premium auto repair workshop.

![Hero](assets/images/hero.jpg)

**Live demo:** https://lionelsinaisinelnikoff.github.io/forge-auto-works

## Features

- Clean, professional dark industrial design
- Fully responsive (mobile-first)
- Interactive image gallery with lightbox + keyboard navigation
- Working contact form (simulated submission with success toast)
- Multiple smooth-scroll CTAs
- 7 custom AI-generated images (via Grok Imagine)
- No frameworks or build steps — pure HTML + Tailwind CDN + vanilla JS

## Tech Stack

- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome icons
- Custom CSS for polish

## Local Development

Just open `index.html` in any browser, or serve it:

```bash
# Python
python3 -m http.server 8000

# or
npx serve .
```

Then visit `http://localhost:8000`.

## Images

All photography was generated using Grok's Imagine model. No real vehicles, people, or locations are depicted.
